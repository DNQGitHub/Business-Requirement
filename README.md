# How to run

## Build Project

npm run build

## Migrate to latest

npm run migrate:latest

## Make new migration

npm run migrate:make <migaration_name>

## Rollback previous migrations

npm run migrate:rollback

## Lint

npm run lint

## Start ( Do this after build )

npm run start

## Start Develop ( No need to build )

npm run start:dev
