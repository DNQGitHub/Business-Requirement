# How to run

## 1) Install dependent package

npm install

## 2) Build Project

npm run build

## 3) Migrate to latest

npm run migrate:latest

## 4) Start

npm run start

# Development Scripts

## Make new migration

npm run migrate:make <migration_name>

## Rollback previous migration

npm run migrate:rollback

## Start Develop ( No need to build )

npm run start:dev

## Lint

npm run lint
