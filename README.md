# How to run

## 1) Install dependencies

```
npm install
```

## 2) Build Project

```
npm run build
```

## 3) Create Database

```
npm run db:create
```

## 4) Migrate to latest

```
npm run migrate:latest
```

## 5) Start

```
npm run start
```

# Development Scripts

## Make new migration

```
npm run migrate:make <migration_name>
```

## Rollback previous migration

```
npm run migrate:rollback
```

## Start Develop With Nodemon

```
npm run start:dev
```

## Lint

```
npm run lint
```
